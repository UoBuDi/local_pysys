# 宝塔面板 WEB 管理界面部署说明 - 数据管理系统（内网部署版）

## 部署包清单

| 压缩包 | 大小 | 内容 |
|--------|------|------|
| `pysys-backend.zip` | ~0.07 MB | 后端 Python 源码 + 配置文件 |
| `pysys-frontend.zip` | ~5.31 MB | 前端构建产物（dist-pro） |

---

## 一、部署架构

```
客户端浏览器
    │
    ▼
Nginx (80)
    ├── /              → 前端静态文件 (pysys-frontend)
    ├── /api/*         → 反向代理 → FastAPI 后端 (8000)
    ├── /ws/*          → WebSocket 代理 → FastAPI 后端 (8000)
    └── /static/*      → 反向代理 → FastAPI 后端 (8000)
```

---

## 二、服务器目录规划

| 目录 | 用途 |
|------|------|
| `/www/wwwroot/pysys-frontend/` | 前端静态文件 |
| `/www/wwwroot/pysys-backend/` | 后端 Python 项目 |
| `/www/wwwroot/pysys-backend/backend/` | 后端代码 |
| `/www/wwwroot/pysys-backend/logs/` | 后端日志 |

---

## 三、环境准备（宝塔 WEB 界面操作）

### 3.1 安装必要软件

**路径：宝塔面板 → 软件商店**

1. **Nginx**
   - 搜索「Nginx」→ 点击「安装」
   - 选择版本 1.22+ → 极速安装 → 提交

2. **Python 项目管理器**
   - 搜索「Python项目管理器」→ 点击「安装」
   - 安装完成后点击「设置」进入管理界面

3. **MySQL**（如数据库在本机）
   - 搜索「MySQL」→ 点击「安装」
   - 选择版本 5.7 或 8.0 → 极速安装 → 提交

### 3.2 安装 Python 3.10.12

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置**

1. 点击「版本管理」标签页
2. 点击「安装版本」
3. 选择 Python 版本：`3.10.12`
4. 点击「安装」
5. 等待安装完成（约 5-10 分钟）

## 四、部署后端（宝塔 WEB 界面操作）

### 4.1 上传后端部署包

**路径：宝塔面板 → 文件**

1. 导航到 `/www/wwwroot/` 目录
2. 点击「新建目录」→ 输入 `pysys-backend` → 确定
3. 进入 `pysys-backend` 目录
4. 点击「上传」→ 选择 `pysys-backend.zip`
5. 上传完成后，右键点击 `pysys-backend.zip` → 「解压」
6. 解压后目录结构：

```
/www/wwwroot/pysys-backend/
├── backend/
│   ├── main.py
│   ├── config.py
│   ├── config.ini          ← 需修改数据库配置
│   ├── database.py
│   ├── models.py
│   ├── requirements.txt
│   ├── migrations/
│   └── ...
├── logs/                   ← 日志目录
├── start.sh                ← 启停脚本
├── pysys-backend.service   ← Systemd 配置
└── nginx_pysys.conf        ← Nginx 配置参考
```

### 4.2 修改数据库配置

**路径：宝塔面板 → 文件 → /www/wwwroot/pysys-backend/backend/**

1. 找到 `config.ini` 文件
2. 右键 → 「编辑」
3. 修改以下配置项：

```ini
[REMOTE_DB]
host = <远程数据库IP>
port = <远程数据库端口>
user = <用户名>
password = <密码>
database = <数据库名>
charset = utf8mb4

[LOCAL_DB]
host = 127.0.0.1
port = 3306
user = <本地数据库用户名>
password = <本地数据库密码>
database = system_db
charset = utf8mb4

[USER_DB]
host = 127.0.0.1
port = 3306
user = <用户数据库用户名>
password = <用户数据库密码>
database = system_db
```

4. 点击「保存」

### 4.3 创建 Python 项目

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置**

1. 点击「项目管理」标签页
2. 点击「添加项目」
3. 填写项目信息：

| 配置项 | 值 |
|--------|-----|
| 项目名称 | `pysys-backend` |
| Python版本 | `3.10.12` |
| 框架 | `FastAPI` |
| 项目路径 | `/www/wwwroot/pysys-backend/backend` |
| 启动方式 | `uvicorn` |
| 启动文件/文件夹 | `main.py` |
| 端口 | `8000` |
| 运行用户 | `www` |
| 开机启动 | ✅ 勾选 |

4. 点击「提交」

### 4.4 安装项目依赖

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置 → 项目管理**

1. 找到 `pysys-backend` 项目
2. 点击「模块」按钮
3. 点击「安装模块」
4. 选择「从 requirements.txt 安装」
5. 等待依赖安装完成

**或使用终端安装：**

**路径：宝塔面板 → 终端**

```bash
cd /www/wwwroot/pysys-backend/backend
/www/server/pyporject/pysys-backend/bin/pip install -r requirements.txt
```

### 4.5 启动后端服务

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置 → 项目管理**

1. 找到 `pysys-backend` 项目
2. 点击「启动」按钮
3. 状态变为「运行中」表示启动成功

### 4.6 初始化数据库（首次部署）

**路径：宝塔面板 → 终端**

```bash
cd /www/wwwroot/pysys-backend/backend
/www/server/pyporject/pysys-backend/bin/python init_statistics_tables.py
/www/server/pyporject/pysys-backend/bin/python init_scheduling_tables.py
```

## 六、配置 Nginx 网站（宝塔 WEB 界面操作）

### 6.1 创建网站

**路径：宝塔面板 → 网站**

1. 点击「添加站点」
2. 填写信息：

| 配置项 | 值 |
|--------|-----|
| 域名 | `内网IP` 或 `内网域名`（如 `192.168.1.100`） |
| 根目录 | `/www/wwwroot/pysys-frontend` |
| FTP | 不创建 |
| 数据库 | 不创建 |
| PHP版本 | 纯静态 |

3. 点击「提交」

### 6.2 配置 Nginx 反向代理

**路径：宝塔面板 → 网站 → 点击站点名 → 设置**

1. 点击左侧「配置文件」
2. 将整个 `server {}` 块替换为以下内容：

```nginx
server {
    listen 80;
    server_name 192.168.1.100;

    root /www/wwwroot/pysys-frontend;
    index index.html;

    gzip on;
    gzip_min_length 1k;
    gzip_comp_level 6;
    gzip_types text/plain application/javascript application/x-javascript text/css application/json text/javascript application/xml;
    gzip_vary on;
    gzip_disable "MSIE [1-6]\.";

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_connect_timeout 60s;
        proxy_read_timeout 120s;
        proxy_send_timeout 60s;
        proxy_buffering off;
    }

    location /ws/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 86400s;
        proxy_send_timeout 86400s;
    }

    location /static/ {
        proxy_pass http://127.0.0.1:8000/static/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    access_log /www/wwwlogs/pysys.access.log;
    error_log /www/wwwlogs/pysys.error.log;
}
```

3. 将 `192.168.1.100` 替换为实际内网 IP 或域名
4. 点击「保存」

### 6.3 重载 Nginx 配置

**路径：宝塔面板 → 软件商店 → Nginx → 设置**

1. 点击「重载配置」或「重启」

## 七、防火墙配置（宝塔 WEB 界面操作）

### 7.1 宝塔面板防火墙

**路径：宝塔面板 → 安全**

1. 确保「放行端口」列表中有：
   - `80`（HTTP）
   - `8888`（宝塔面板，如需远程访问）
2. **不要放行** `8000` 端口（后端仅内部访问）

### 7.2 内网访问说明

- 内网部署无需配置云服务器安全组
- 确保客户端与服务器在同一内网网段
- 如有内网防火墙设备，需放行 80 端口

## 八、验证部署

### 8.1 检查后端服务

**路径：宝塔面板 → 终端**

```bash
curl http://127.0.0.1:8000/api/config/
```

应返回 JSON 格式的配置信息。

### 8.2 检查前端页面

浏览器访问 `http://192.168.1.100`（替换为实际内网 IP），应看到登录页面。

### 8.3 检查 API 代理

浏览器访问 `http://192.168.1.100/api/config/`，应返回与后端直接访问相同的 JSON。

### 8.4 检查 WebSocket

登录系统后，打开浏览器开发者工具（F12）→ Network → WS，应看到 WebSocket 连接已建立。

## 九、日常维护（宝塔 WEB 界面操作）

### 9.1 查看后端日志

**路径：宝塔面板 → 文件**

- 后端应用日志：`/www/wwwroot/pysys-backend/logs/app.log`
- Uvicorn 日志：`/www/wwwroot/pysys-backend/logs/uvicorn.log`

或使用终端：
```bash
tail -f /www/wwwroot/pysys-backend/logs/app.log
```

### 9.2 重启后端服务

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置 → 项目管理**

找到 `pysys-backend` 项目 → 点击「重启」

### 9.3 更新前端

1. 本地执行 `pnpm build:pro` 构建新版本
2. 打包 `dist-pro/` 为 `pysys-frontend.zip`
3. 宝塔面板 → 文件 → 上传到 `/www/wwwroot/pysys-frontend/`
4. 解压覆盖旧文件
5. 无需重启任何服务

### 9.4 更新后端

1. 打包新的后端代码
2. 宝塔面板 → 文件 → 上传到 `/www/wwwroot/pysys-backend/backend/`
3. 如有新依赖：Python项目管理器 → 模块 → 安装模块
4. 如有数据库变更：终端执行 `python run_migration.py`
5. Python项目管理器 → 重启项目

## 十、常见问题排查

### 10.1 后端启动失败

**路径：宝塔面板 → 软件商店 → Python项目管理器 → 设置 → 项目管理**

1. 点击项目「日志」按钮查看错误信息
2. 常见原因：
   - 依赖未安装完整 → 重新安装 requirements.txt
   - 端口被占用 → 检查 8000 端口是否被其他程序占用
   - 配置文件错误 → 检查 config.ini 格式

### 10.2 前端页面空白

1. 检查 Nginx 配置中的 `root` 路径是否正确
2. 检查 `index.html` 是否存在
3. 检查浏览器控制台是否有 JS 错误

### 10.3 API 请求 502 错误

1. 检查后端服务是否运行：Python项目管理器 → 查看状态
2. 检查 Nginx 反向代理配置是否正确
3. 检查后端日志是否有错误

### 10.4 WebSocket 连接失败

1. 检查 Nginx 的 `/ws/` location 配置是否正确
2. 确保包含 `proxy_set_header Upgrade` 和 `Connection "upgrade"`
3. 内网环境使用 `ws://` 协议（非 `wss://`）

### 10.5 内网无法访问

1. 检查服务器 IP 地址是否正确
2. 检查客户端与服务器是否在同一网段
3. 检查服务器防火墙是否放行 80 端口
4. 尝试在服务器本地访问 `curl http://127.0.0.1` 排查
