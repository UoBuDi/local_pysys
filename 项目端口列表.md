# 项目端口列表

## 1. 后端服务端口
- **端口号**：8000
- **用途**：FastAPI后端服务
- **配置位置**：
  - 启动命令：`uvicorn main:app --reload --port 8000 --app-dir backend`
  - 代码中：无硬编码，通过命令行参数指定
- **当前状态**：正在运行

## 2. 前端服务端口
- **端口号**：4000
- **用途**：Vite前端开发服务器
- **配置位置**：
  - 启动命令：`pnpm dev --port 4000`
  - 代码中：无硬编码，通过命令行参数指定
- **当前状态**：正在运行

## 3. 数据库端口
- **端口号**：3306
- **用途**：MySQL数据库连接
- **配置位置**：
  - 配置文件：`backend/config.ini`
  - 代码中：默认值为3306
- **当前状态**：配置中使用

## 4. WebSocket端口
- **端口号**：8000（与后端服务相同）
- **用途**：实时同步进度和日志推送
- **配置位置**：
  - 前端：动态从`API_BASE_URL`获取
  - 后端：与HTTP服务共享端口
- **当前状态**：已修复，可正常连接

## 5. 其他端口
- **端口号**：5000
  - 用途：前端SyncConfig.vue中批量大小的最大值
  - 配置位置：`src/views/SystemTools/SyncConfig.vue:160`

- **端口号**：443
  - 用途：HTTPS默认端口，在WebSocket URL构建中使用
  - 配置位置：`src/views/SystemTools/SyncControl.vue:367`

- **端口号**：80
  - 用途：HTTP默认端口，在WebSocket URL构建中使用
  - 配置位置：`src/views/SystemTools/SyncControl.vue:367`

## 配置文件端口

### 1. 后端配置文件
- **文件路径**：`backend/config.ini`
- **端口配置**：
  ```ini
  [REMOTE_DB]
  port = 3306
  
  [LOCAL_DB]
  port = 3306
  
  [USER_DB]
  port = 3306
  ```

### 2. 前端配置文件
- **文件路径**：`vue-element-plus-admin-master/.env`
- **端口配置**：
  ```env
  VITE_API_BASE_PATH=http://localhost:8000
  ```

## 启动命令

```bash
uvicorn main:app --reload --port 8000 --app-dir backend
```

### 2. 前端启动命令

## 注意事项

1. 确保所有服务使用的端口不冲突
2. 修改端口后，需要同步更新相关配置文件
3. WebSocket端口应与API端口保持一致
4. 数据库端口应与实际数据库服务端口匹配
5. 前端开发服务器端口可根据需要调整

## 故障排除

- **端口被占用**：使用`netstat -ano | findstr :端口号`查看占用进程，使用`taskkill /F /PID 进程ID`终止进程
- **WebSocket连接失败**：检查WebSocket URL是否正确，确保与API使用相同的端口
- **数据库连接失败**：检查数据库端口、用户名和密码是否正确
